You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /covert -- coverts from miles to km or km to miles.
    example: http://localhost:9000/convert?value=15&from=miles&to=km
  • /multiplied squares - multiplies the square of two integers and outputs the closest perfect square.
    example:http://localhost:9000/multiply_squares?num1=2&num2=3
